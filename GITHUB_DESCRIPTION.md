## Full Description (for README or GitHub About section):

**Mobile Mart** is a modern, full-stack e-commerce platform specializing in mobile devices and accessories. The application features a responsive Next.js frontend with TypeScript and Tailwind CSS, backed by a Node.js/Express RESTful API.

### Key Technologies:
- Frontend: Next.js 16, React 19, TypeScript, Tailwind CSS 4
- Backend: Node.js, Express.js, MongoDB
- Features: Product catalog, shopping cart, user authentication, product details, search & filters
